# Brain Cloud Archive

## 概要 (Overview)
[日本語]
「Brain Cloud」プロジェクトに関する機密アーカイブへのアクセスシミュレーションです。
90年代のレトロなOS環境を模したインターフェースを通じて、プロジェクトの断片的な資料を閲覧できます。

[English]
A simulated access portal to the classified archives of the "Brain Cloud" project.
Experience an interface modeled after a 90s retro OS, where users can browse fragments of the project's historical data.
